## 🔧 技术栈总结

### 后端技术栈
```
Python 3.9
├── FastAPI (Web框架)
├── SQLAlchemy (ORM)
├── Pydantic (数据验证)
├── PostgreSQL (数据库)
├── Redis (缓存)
├── Alembic (数据库迁移)
├── Pytest (测试框架)
└── Uvicorn (ASGI服务器)
```

### 前端技术栈
```
微信小程序
├── 原生小程序框架
├── WeUI (UI组件库)
├── wx.request (网络请求)
├── wx.uploadFile (文件上传)
└── 小程序云开发 (可选)
```

### 基础设施技术栈
```
Docker & Docker Compose
├── Nginx (反向代理)
├── PostgreSQL (数据库)
├── Redis (缓存)
├── Prometheus (监控)
├── Grafana (可视化)
└── Let's Encrypt (SSL证书)
```
